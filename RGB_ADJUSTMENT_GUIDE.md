# RGB通道调节功能使用指南

## 功能说明

RGB通道调节功能允许您独立调整图像的红色(R)、绿色(G)和蓝色(B)通道，实现精细的色彩控制。

## 使用方法

1. **启动项目**
   ```bash
   cd /Users/herbal/code/src/image-editor
   pnpm dev
   ```
   服务器将在 http://localhost:5173 启动

2. **导入图像**
   - 点击"导入图片"按钮选择图片
   - 或直接拖拽图片到画布区域

3. **调节RGB通道**
   在左侧控制面板中找到"RGB通道调节"部分：
   - **红色(R)滑块**：调整范围 -100% 到 +100%
     - 负值：减少红色通道强度
     - 正值：增加红色通道强度
   - **绿色(G)滑块**：调整范围 -100% 到 +100%
     - 负值：减少绿色通道强度
     - 正值：增加绿色通道强度
   - **蓝色(B)滑块**：调整范围 -100% 到 +100%
     - 负值：减少蓝色通道强度
     - 正值：增加蓝色通道强度

## 调节效果示例

### 增强暖色调

## 技术实现

### 算法原理
RGB调节通过直接修改图像的RGB通道值来实现色彩调整：

```typescript
// 对于每个像素
R_new = clamp(R_old + (adjustment_R / 100) * 255, 0, 255)
G_new = clamp(G_old + (adjustment_G / 100) * 255, 0, 255)
B_new = clamp(B_old + (adjustment_B / 100) * 255, 0, 255)
```

### 特点
1. **独立调节**：三个通道完全独立，互不影响
2. **实时预览**：调节滑块时实时显示效果（100ms防抖）
3. **非破坏性**：所有调节基于原始图像，可随时重置
4. **可组合**：可与其他滤镜（亮度、对比度等）组合使用

## 注意事项

1. **调节顺序**：RGB调节在基础调整（亮度、对比度、饱和度）之后应用
2. **重置功能**：点击"重置"按钮可将所有调节恢复到初始状态
3. **历史记录**：调节后点击"烘焙"可保存当前状态到历史记录
4. **导出**：调节后的图像可导出为PNG或JPEG格式

## 与其他功能的配合

### 推荐工作流程

2. 调整视图模式（适配/1:1/填充）
3. 应用基础调整（亮度、对比度、饱和度）
4. 应用RGB通道调节进行精细色彩控制

6. 导出最终图像

## 常见问题

**Q: RGB调节和饱和度调节有什么区别？**
A: 饱和度调节影响整体色彩鲜艳度，而RGB调节可以独立控制每个颜色通道，实现更精细的色彩控制。

**Q: 为什么调节后图像看起来不自然？**
A: 建议使用较小的调节幅度（±20%以内），过大的调节可能导致色彩失真。

**Q: 可以同时调节多个通道吗？**
A: 可以，三个通道可以同时调节，效果会叠加。
